# 📚 Library Management System

A modern full-stack Library Management System developed using PHP and MySQL.
This project includes a responsive admin dashboard for managing books, users and loan records with database integration and CRUD operations.

🌐 Live Demo: https://merve-library.free.nf

---

# ✨ Features

* Modern admin dashboard
* Book management system
* User management module
* Loan & borrowing tracking
* Responsive interface design
* MySQL database integration
* CRUD operations
* Clean UI/UX structure
* Full-stack architecture

---

# 🖥️ Dashboard Modules

## 📖 Books

* Add new books
* List all books
* Track authors and categories
* Manage publication details

## 👤 Users

* Register library members
* Store user information
* Manage active/passive status

## 🔄 Loans

* Track borrowed books
* Manage return dates
* Monitor active loans
* Loan status management

# 💻 Local Setup with XAMPP

## 1. Start XAMPP Services

Start:

* Apache
* MySQL

from the XAMPP Control Panel.

---

## 2. Move Project Folder

Copy the project folder into:

```text
C:\xampp\htdocs\library-management-system
```

---

## 3. Open phpMyAdmin

```text
http://localhost/phpmyadmin
```

---

## 4. Create Database

Create a database named:

```text
library_management
```

---

## 5. Import SQL File

Import:

```text
database/library_management.sql
```

---

## 6. Run the Project

Open in browser:

```text
http://localhost/library-management-system
```
